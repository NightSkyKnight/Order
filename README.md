# Order

## 环境

 1. Windows7/windows10
 2. python3.7

针对某系统后台的刷单程序
一级域名部分进行了保密处理

程序实现登陆获取cookie，并使用cookie进行刷单，刷出的内容展示到TXT文件里；<br>

按Ctrl+C可以暂停程序；
程序在12:00到14：00自动暂停；
